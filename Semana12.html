Semana 12

Descripción

Esta semana nos enfocamos en el desarrollo de aplicaciones backend utilizando Jakarta Server Pages (JSP). Explora cómo JSP permite la creación de aplicaciones web dinámicas mediante el uso de scriptlets, declaraciones y expresiones, todo sin la necesidad de servlets o frameworks como Spring. El objetivo fue construir aplicaciones web funcionales con la última versión de Tomcat y JDK.

Tópicos Aprendidos

Definiciones

Jakarta Server Pages (JSP) es una tecnología de scripting del lado del servidor para construir aplicaciones web dinámicas. JSP permite la integración con todas las APIs de Java, incluyendo JDBC para la interacción con bases de datos. Los elementos clave en JSP son scriptlets para insertar código Java, declaraciones para definir variables y métodos, y expresiones para evaluar y mostrar datos.

Procedimientos

  1. Creación de archivos JSP: Utilicé scriptlets para integrar código Java, declaraciones para definir variables y métodos, y expresiones para mostrar datos en las páginas.
  2. Desarrollo de aplicaciones: Construí aplicaciones web completas utilizando solo JSP, sin la integración de servlets o frameworks adicionales, enfocándome en el uso puro de la tecnología.
  3. Implementación: Utilicé Tomcat 11 y JDK-24 para asegurar la compatibilidad y correcto funcionamiento de las aplicaciones JSP desarrolladas.

Resultados

El resultado fue una aplicación web funcional construida exclusivamente con JSP, demostrando cómo los scriptlets, declaraciones y expresiones pueden ser utilizados para crear contenido dinámico. La implementación en un entorno actualizado con Tomcat 11 y JDK-24 garantizó la compatibilidad y el buen funcionamiento de la aplicación.

Reflexión Metacognitiva

Reflexionar sobre el uso de JSP me hizo apreciar la importancia de comprender y manejar las técnicas de scripting del lado del servidor para crear aplicaciones web completas. Aunque JSP ofrece un enfoque directo, también resalta la necesidad de considerar frameworks y servlets en contextos más complejos.

¿Qué aprendí?

Aprendí a utilizar JSP para crear contenido web dinámico sin depender de servlets o frameworks adicionales, enfocándome en los elementos fundamentales de JSP para desarrollar aplicaciones web completas.

¿Cómo aprendí?

Aprendí mediante la práctica directa en proyectos y la realización de ejercicios centrados en el uso de JSP. Cada ejercicio me permitió explorar y aplicar las capacidades de JSP en la creación de aplicaciones web.
