Semana 9

Descripción

En esta semana, me enfrenté a un reto emocionante: la evaluación parcial en la asignatura de Desarrollo de Aplicaciones Web. Este desafío incluyó la creación de una ruleta interactiva utilizando HTML, CSS y JavaScript, y el desarrollo de una aplicación en React con componentes anidados. Cada proyecto representó una oportunidad para profundizar en mis habilidades y explorar nuevas dimensiones de la programación web.

Tópicos Aprendidos

Definiciones

Aprendí a desarrollar una Quiz manipulando el DOM con JavaScript puro. Además, me sumergí en el mundo de React, creando una aplicación con cuatro componentes anidados, donde la gestión y actualización de datos se realiza de manera fluida desde el componente principal al más profundo.

Procedimientos

  1. Desarrollo de la Ruleta: Implementé una ruleta interactiva usando JavaScript puro para manipular el DOM y seleccionar un elemento de manera aleatoria.
  2. Creación de Aplicación en React: Construí una aplicación con cuatro componentes anidados, gestionando el estado y la actualización de datos desde el componente principal hacia los más profundos.
  3. Interacción y Actualización: Implementé funciones para actualizar la interfaz de usuario en respuesta a las interacciones del usuario en ambos proyectos.

Resultados

El resultado de esta semana fue una aplicación web completamente funcional, que incluye una ruleta interactiva y una aplicación en React demostrando el manejo eficiente de estados y datos entre componentes. Esta experiencia no solo reforzó mi comprensión de la manipulación del DOM, sino también la arquitectura de componentes en React, elevando mis habilidades en desarrollo web.

Reflexión Metacognitiva

Reflexionar sobre la creación de interfaces interactivas y la estructura de aplicaciones en React me hizo apreciar la importancia de mantener un código claro y manejable. Aprendí cómo una buena estructura y una correcta gestión del estado pueden transformar la experiencia del usuario y hacer que las aplicaciones sean más eficientes y fáciles de mantener.

¿Qué aprendí?

Aprendí a desarrollar componentes interactivos y a gestionar el estado en React, así como a crear interfaces web que responden de manera dinámica a las interacciones del usuario. Este conocimiento me ha proporcionado una base sólida para futuros proyectos en desarrollo web.

¿Cómo aprendí?

Mi aprendizaje se consolidó a través de la práctica en proyectos desafiantes y la realización de ejercicios enfocados en el desarrollo de componentes y la gestión del estado en React. Cada tarea y cada implementación me llevaron a una comprensión más profunda y una ejecución más eficiente de mis habilidades de programación.
