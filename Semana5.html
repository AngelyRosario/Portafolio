Semana 5

Descripción

Durante esta semana, me aventuré en el desarrollo de funcionalidades interactivas para páginas web, explorando las posibilidades que ofrece HTML y JavaScript. El enfoque principal fue la implementación de eventos y la manipulación del DOM, herramientas que permiten transformar la experiencia del usuario y dar vida a las páginas web. Cada interacción se convirtió en una oportunidad para mejorar la conexión entre el usuario y el contenido, haciendo que cada página se sintiera más dinámica y personalizada.

Tópicos Aprendidos

Definiciones

La manipulación del DOM en JavaScript me permitió adentrarme en el corazón de las páginas web, interactuando directamente con los elementos HTML y respondiendo a las acciones del usuario. Aprendí a cambiar el contenido de una página, actualizar valores de manera dinámica y gestionar eventos de clic para hacer la experiencia más rica y envolvente.

Procedimientos

  1. Mostrar la fecha actual: Implementé funciones para mostrar la fecha en tiempo real en la página.
  2. Actualizar contenido: Creé funcionalidades para cambiar el contenido de elementos de texto de manera dinámica.
  3. Calcular sumas: Desarrollé scripts para calcular sumas basadas en entradas del usuario.
  4. Mostrar/ocultar datos: Añadí funciones para mostrar y ocultar datos en tablas, mejorando la interacción y la visualización.

Resultados

La implementación de estas funcionalidades resultó en una interacción más fluida y enriquecedora para el usuario. Cada cambio en el contenido y cada respuesta a un clic se tradujo en una experiencia más envolvente, donde la página web se convirtió en un entorno más receptivo y dinámico.

Reflexión Metacognitiva

Reflexionar sobre esta semana me permitió apreciar cómo la manipulación del DOM y el manejo de eventos pueden transformar la forma en que los usuarios interactúan con las páginas web. Estos conocimientos no solo mejoran la funcionalidad, sino que también enriquecen la experiencia del usuario, haciendo que cada interacción sea significativa y memorable.

¿Qué aprendí?

Aprendí a implementar eventos y manipular el DOM para llevar la interacción del usuario en las páginas web a un nuevo nivel. Esta habilidad me permite crear experiencias más dinámicas y adaptadas a las necesidades del usuario.

¿Cómo aprendí?

Mi aprendizaje se forjó a través de la práctica en proyectos y la realización de ejercicios enfocados en eventos y manipulación del DOM. Cada proyecto y ejercicio fue una oportunidad para experimentar y perfeccionar estas habilidades, profundizando en la forma en que el JavaScript puede transformar la interacción con el usuario.
